# EVM in Rust

## Overview

- Explore the EVM implemented in Rust
- 2 objectives:
  - Learn Rust
  - Learn EVM

## Installation

```bash
git clone git@github.com:rust-blockchain/evm.git
cd evm
cargo build --release --all
```
